# Task 2: User Management System
## Smart Methods Journey - Web and App Development Department

### What is this task?
This project creates a **user management system** using HTML, CSS, JavaScript, and PHP with MySQL database. It's a dynamic website that lets you add users and manage their status with real-time updates.

### How it works
1. Open the website in a browser (requires XAMPP/WAMP)
2. Fill out the form with name and age
3. Click Submit to add user to database
4. View all users in the table below
5. Click Toggle button to change user status
6. Status updates immediately on the webpage

### How to use it
1. **Setup:**
   - Install XAMPP or WAMP
   - Start Apache and MySQL services
   - Create database named `task2`
   - Place PHP file in htdocs folder

2. **Using the System:**
   - Enter name and age in the form
   - Click Submit to add user
   - View users in the table below
   - Use Toggle button to change status
   - Status changes from Inactive (0) to Active (1)

### Example result
A working user management system that:
- Adds users to database when form is submitted
- Shows all users in a clean table format
- Toggles user status between Active and Inactive
- Updates status immediately when Toggle is clicked
- Stores all data permanently in MySQL database
